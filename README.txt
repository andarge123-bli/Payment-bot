REQUIRED ASSETS — Place these files in this directory before running the bot:

1. template.png
   - Your membership card background image
   - Recommended size: 700 x 450 pixels
   - The bot overlays user name, ID, status, and date on top of it
   - If missing, the bot auto-generates a default dark-blue card

2. Nyala.ttf
   - Amharic/Ethiopic font file for text rendering on the card
   - Found on Windows at: C:\Windows\Fonts\nyala.ttf
   - Alternative: Download "Noto Serif Ethiopic" from Google Fonts
   - If missing, the bot falls back to PIL's default font (no proper Amharic)
